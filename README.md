<h1> raspi</h1>

<p>这是一个本科毕业设计——《基于树莓派的智能家居监控系统》中的一个 django web服务</p>
<p>这个web服务实现了只有管理员才能登录的访问权限,以使只有主人才能访问和控制家居设备
<p>初次使用需要创建django的admin管理员账户：$python manage createsuperuser</p>
<ul>主要用到:
    <li>python 2.7.13<li>
    <li>django 1.11</li>
    <li>bootstrap3.3.7</li>
</ul>
